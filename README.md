# APIFOME

NPM package that uses the [API](https://apifome.online)

---

JS INSTALL

      $ npm install apifome

TS INSTALL

      $ npm install apifome

HOW TO USE IN JS

    const {Food} = require('apifome')

    let food = await Food('batata') // Enter a food in string

HOW TO USE IN TS

    import {Food} from 'apifome'
    let food = await Food('batata') // Enter a food in string
